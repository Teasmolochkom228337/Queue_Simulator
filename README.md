## Возможности

- Визуализация очередей в реальном времени
- Случайное распределение новых клиентов
- Вероятностное обслуживание на кассах
- Подсчёт общего количества обслуженных клиентов

## Технологии

- C++17
- Стандартная библиотека:
  - `thread` - для задержек визуализации
  - `random` - для генерации случайных событий
  - `chrono` - для управления временем
  - `vector` - для хранения очередей

## Установка и запуск

### Требования
- Компилятор с поддержкой C++17 (GCC 7+, Clang 5+, MSVC 2017+)
- CMake (опционально)
